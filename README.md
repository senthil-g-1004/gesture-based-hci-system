# Gesture-Based Human–Computer Interaction System

This project allows users to control mouse movement and click actions using hand gestures captured through a webcam.

The system tracks hand landmarks in real time and maps finger movements to mouse operations, enabling touchless interaction with the computer.

## Technologies Used
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## Files Included
- main.py – Main implementation file
- requirements.txt – Required Python libraries
- INSTRUCTIONS.md – How to install and use the project
